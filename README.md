# User-Management-System

Project Overview

This is a full-stack CRUD (Create, Read, Update, Delete) application built with React for the frontend and Node.js + Express for the backend, using MongoDB as the database.
It allows users to create, view, update, and delete user records in MongoDB.

Features

Add new users
Read all users
Update existing users
Delete users
RESTful API powered by Express.js
MongoDB database for persistent storage
Responsive React frontend

Technologies Used

Frontend: React, JSX, Axios
Backend: Node.js, Express, CORS
Database: MongoDB
Deployment: Render, GitHub
